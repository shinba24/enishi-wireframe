ご相談・費用について

Q.相談は本当に無料ですか？

+

A.はい、ご相談は完全無料です。LINEでのご相談から、ご成約まで一切費用はかかりません。お気軽にご連絡ください。

Q.遠方でも対応してもらえますか？

+

A.全国どこからでもご相談いただけます。オンライン・LINEでの対応が可能ですので、お気軽にご連絡ください。

Q.しつこい営業はありますか？

+

A.一切ありません。お客様のペースに合わせてご対応いたします。「まだ検討中」という段階でも大歓迎です。

Q.LINEで相談した場合、どのくらいで返信がもらえますか？

+

A.営業時間内（9:00〜18:00）であれば、基本的に当日中にご返信いたします。お急ぎの場合はお電話でもご対応可能です。

Q.仲介手数料はいくらですか？

+

A.宅地建物取引業法に基づく上限額の範囲内で設定しております。詳細はご相談内容によって異なりますので、お気軽にお問い合わせください。

賃貸について

Q.希望条件を伝えれば物件を探してもらえますか？

+

A.はい、ご希望の条件（エリア・家賃・間取り・こだわり条件など）をお伝えいただければ、専門スタッフが最適な物件をご提案します。

Q.内見はオンラインでも可能ですか？

+

A.はい、オンライン内見にも対応しています。遠方にお住まいの方や、忙しくて現地に行けない方もお気軽にご相談ください。

Q.法人契約にも対応していますか？

+

A.はい、法人契約にも対応しております。社宅・事務所・店舗など、法人のご相談もお気軽にどうぞ。

売買について

Q.住宅ローンの相談もできますか？

+

A.はい、住宅ローンのご相談にも対応しています。提携金融機関のご紹介や、返済シミュレーションのご提案も行っています。

Q.投資用物件の購入相談もできますか？

+

A.はい、投資用物件の購入・収益シミュレーション・運用相談にも対応しています。初めての不動産投資の方もお気軽にご相談ください。

Q.購入から引き渡しまでどのくらいかかりますか？

+

A.物件の状況や住宅ローンの審査状況によって異なりますが、一般的には2〜3ヶ月程度が目安です。詳しくはご相談時にご説明します。

売却・査定について

Q.売却査定はどのくらいで回答もらえますか？

+

A.簡易査定は最短即日でご回答いたします。詳細査定は物件の状況により数日いただく場合がございます。

Q.相続した不動産の売却も相談できますか？

+

A.はい、相続不動産の売却・活用・賃貸化など、複雑な状況にも対応しています。税理士・司法書士との連携サポートも行っています。

Q.売却活動中も住み続けることはできますか？

+

A.はい、売却活動中も引き続きお住まいいただけます。内見の日程調整などもご相談に応じて柔軟に対応します。

Q.売却にかかる費用はどのくらいですか？

+

A.仲介手数料（売却価格の3%+6万円+消費税が上限）のほか、登記費用・税金などがかかります。詳細はご相談時にご説明します。
